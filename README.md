# Catálogo de Produtos API 📦

API REST construída em .NET 8 com C# para o gerenciamento de Usuários, Produtos e Subprodutos.

## 🏗️ Arquitetura
Este projeto foi desenvolvido aplicando os conceitos de **Domain-Driven Design (DDD)** e dividindo as responsabilidades em 4 camadas principais:
- **Domain:** Entidades e regras de negócio isoladas.
- **Application:** Casos de uso e DTOs.
- **Infrastructure:** Acesso a dados e repositórios.
- **API:** Controladores e endpoints REST.

## 🚀 Tecnologias Utilizadas
- .NET 8
- C#
